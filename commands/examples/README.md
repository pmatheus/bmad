## Purpose

These examples were extracted from the original workflow files (1,600+ lines each) during condensation to keep the main workflow commands focused and concise (~250-350 lines).

The examples provide:
- Real-world scenarios showing workflow execution
- Different project types (SaaS, Healthcare, Mobile, etc.)
- Various outcomes (success, blocked, changes requested)
- Step-by-step walkthroughs
- Common patterns and edge cases

## Usage

Reference these examples when:
- Learning a new workflow for the first time
- Troubleshooting workflow execution
- Understanding workflow outcomes
- Training team members
- Preparing for complex scenarios

## Note

Examples are extracted from original files dated November 2025. For the most current examples, consult the official BMAD documentation or the original pre-condensation command files in the BACKUP directory.
